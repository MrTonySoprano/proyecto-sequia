\# Proyecto Django – Gestión de Sequía



Este proyecto corresponde a una aplicación web desarrollada en \*\*Django\*\*, enfocada en la gestión de información asociada a la sequía. El sistema permite administrar registros, sectores y datos asociados, facilitando el análisis y administración de recursos hídricos.



---



\## Requisitos del proyecto



\- Python 3.x  

\- Django 4.x o superior  

\- Sistema operativo Windows, Linux o macOS  

\- Opcional: archivo `.env` para credenciales
